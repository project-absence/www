This module will take screenshots of newly discovered domains.

## Configuration

| Parameter      | Description                                                                                                                           | Default |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------- | ------- |
| `chrome_path`  | Path for Chrome or Chromium. If unspecified, it will try to automatically detect a suitable binary.                                   |         |
| `save_as_file` | Whether the screenshots should be saved as separate file. If false, it will base64 encode the screenshot and save it in the JSON file | `false` |
